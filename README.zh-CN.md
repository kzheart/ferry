# Ferry

[English](README.md)

Ferry 是面向 Claude Code、Codex CLI 和 OpenCode 的桌面会话管理工具。它将不同 Coding Agent 的本地会话集中到一个界面中，并帮助你在 Agent 之间延续有价值的工作上下文。

## 功能

- 统一浏览和搜索 Claude Code、Codex CLI 与 OpenCode 的本地会话。
- 在一致的界面中查看对话、工具调用、推理摘要、元数据和会话结构。
- 将会话迁移到另一个 Coding Agent，并保留继续工作所需的上下文。
- 删除对话轮次、改写消息和裁剪过长的工具输出。
- 应用修改前预览结果，并通过备份安全地编辑和恢复会话。
- 在对应 Agent 中直接继续迁移或编辑后的会话。
- 检测已安装的 Agent、可用模型和本地会话数据。
- 在应用内检查更新、查看下载进度，并在确认后安装更新。

Ferry 支持 Apple Silicon Mac 和 64 位 Windows。
